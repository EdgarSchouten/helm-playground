# helm-playground

Playground for helm

This project uses docker desktop for mac for local kubernetes cluster.

### Pre-requisite

- [Docker desktop for mac](https://docs.docker.com/docker-for-mac/install/)
- [Helm](https://helm.sh/docs/intro/install/) `v3.1.2`
